# 07. Industry Package 设计

## 文档状态

**状态**：占位草案  
**范围**：行业包插件结构、数据源依赖、行业 Agent、评分规则、市场映射  
**当前约定**：行业包也是插件，类型为 `industry`

## 后续需要讨论的问题

1. 行业包是否必须声明依赖的 source 插件？
2. 行业包是否允许自带 Agent workflow，还是只能配置核心模板？
3. 行业知识图谱放在插件内，还是统一存储在数据库？
4. 行业包如何声明关注市场、标的、交易工具和 executor 偏好？
5. 行业包评分规则是否允许自定义？
6. 行业包是否需要自己的测试样例和评估集？
7. 行业包配置中的关键词、账号、行业逻辑是否默认私有？
8. 多个行业包同时命中同一事件时如何合并或冲突处理？

## 暂不决策

- 石油行业包具体规则。
- 半导体行业包具体规则。
- 行业知识图谱格式。
- 行业评分公式。
