# AGENTS.md

## 定位

- `packages/adapters` 是外部系统适配器的预留 package。
- 该目录面向数据源、市场数据、通知、交易通道等外部能力适配。
- 当前目录尚未落地实现，只保留 package 边界。

## 行为约束

- 不在没有 issue、OpenSpec 或设计文档真源支撑的情况下提前添加真实外部服务集成。
- 适配器不应把第三方 API 的响应结构直接泄漏成 core 的领域模型。
- 外部凭据必须通过环境变量或 secret reference 注入，不写入源码或样例配置。
- 外部动作默认应有 mock、dry-run 或禁用模式，避免开发期触发真实外部副作用。
- 外部请求需要有超时、限流、错误摘要和来源记录；不能无限重试。
- 适配器提供给 Agent 或插件的能力应注册为受控工具或 port，不直接暴露无限权限调用。
